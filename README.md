# OS Traffic Simulation Project

A multi-agent traffic simulation system implemented in C using the Raylib library. The system demonstrates operating system fundamentals including concurrent process management, Inter-Process Communication (IPC), signal handling, and synchronization of shared resources.

## 🚀 Compilation and Execution

The project is compiled using the provided Makefile.

* **Milestone 2:** `make milestone2`
* **Milestone 3:** `make milestone3`
* **Milestone 4:** `make milestone4`
